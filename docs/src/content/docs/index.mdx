## Features

<CardGrid>
	<Card title="EVM Client" icon="pencil">
		Execute the EVM directly in your JavaScript code. It's like having anvil in your browser, node, or bun enviornments.
	</Card>
	<Card title="Built as viem transport" icon="add-document">
		Plugs directly into viem and wagmi via a viem transport. Ethers.js provider is also available.
	</Card>
	<Card title="Solidity to TypeScript compiler" icon="setting">
		With tevm you don't need a seperate compiler to generate abis and bytecode from solidity. You can import solidity directly into TypeScript files with ESM imports.
	</Card>
	<Card title="Powerful scripting" icon="setting">
		Tevm can run arbitrary solidity script with advanced features such as JavaScript precompiles
	</Card>
	<Card title="Runs in all environments" icon="open-book">
		Tevm runs in the browser, node.js, and bun
	</Card>
</CardGrid>
